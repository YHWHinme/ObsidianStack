# Main Headline: Five Go Projects to Master the Language

## Sub-Headlines:

* **Headline:**  Conquer Go with a Command-Line To-Do List
    * **Snippet:** Learn fundamental Go concepts by building a terminal-based to-do list application.  This project covers file system I/O, user interface design, and handling multiple commands.  The use of a simple CSV file for data storage minimizes complexity.
    * **Timestamp:** `[00:00:21 - 00:02:37]`

* **Headline:**  Craft a Stateless Go Web API: Calculator Service
    * **Snippet:** Build a fully stateless calculator API to master backend web development in Go. This project emphasizes API implementation without data store concerns, focusing on clean design and common patterns.  An OpenAPI specification is provided for guidance.
    * **Timestamp:** `[00:02:43 - 00:04:07]`

* **Headline:**  Build a Dead Link Detector Web Scraper
    * **Snippet:**  Create a CLI application to scan websites for broken links.  This project teaches web scraping techniques using the `net/http` and `net/html` packages, along with handling redirects and concurrency for improved performance.
    * **Timestamp:** `[00:05:30 - 00:07:43]`

* **Headline:**  Develop a URL Shortener Web App in Go
    * **Snippet:**  Construct a simple web application that shortens URLs using Go's `html/template` package for dynamic data display and `net/http` for redirect handling.  Learn to manage HTTP redirects and handle non-existent URLs.
    * **Timestamp:** `[00:07:48 - 00:09:49]`

* **Headline:**  Create a Currency Converter with a Terminal UI
    * **Snippet:** Build a terminal UI-driven currency converter using the `charm/ui` library.  This project involves interfacing with a third-party API (Open Exchange Rates), securing API keys, and handling currency conversions.
    * **Timestamp:** `[00:09:53 - 00:11:09]`
